.NET Runtime

This repo contains the code to build the .NET runtime, libraries, and shared host (dotnet) installers for all supported platforms. It includes the following components:

What is .NET?

.NET is a free, cross-platform, open source developer platform for building many different types of applications. With .NET, you can use multiple languages, editors, and libraries to build for web, mobile, desktop, games, IoT, and more.

Components

  • CoreCLR - the runtime, JIT compiler and base library for .NET
  • Mono - the runtime for mobile and browser platforms (iOS, Android, WASM)
  • Libraries - the standard class libraries System.*
  • NativeAOT - ahead-of-time compilation

How to Build

Prerequisites

  • Install CMake
  • Install Python 3

Building on Windows

.\build.cmd -subset clr+libs -configuration Release

Building on Linux/macOS

./build.sh -subset clr+libs -configuration Release
