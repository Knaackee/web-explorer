System.Collections.Concurrent > ConcurrentDictionary

ConcurrentDictionary<TKey,TValue> Class

Namespace: System.Collections.Concurrent Assembly: System.Collections.Concurrent.dll

Represents a thread-safe collection of key/value pairs that can be accessed by multiple threads concurrently.

Definition

public class ConcurrentDictionary<TKey,TValue> : IDictionary<TKey,TValue>,
    ICollection<KeyValuePair<TKey,TValue>>, IReadOnlyDictionary<TKey,TValue>
    where TKey : notnull

Remarks

For very large ConcurrentDictionary<TKey,TValue> objects, you can increase the maximum array size to 2 gigabytes (GB) on a 64-bit system by setting the gcAllowVeryLargeObjects configuration element to true in the run-time environment.

All public and protected members of ConcurrentDictionary<TKey,TValue> are thread-safe and may be used concurrently from multiple threads. However, members accessed through one of the interfaces implement, including extension methods, are not guaranteed to be thread safe.

Thread Safety

All public members of ConcurrentDictionary<TKey,TValue> are thread-safe. The dictionary uses fine-grained locking to allow concurrent reads and a linear number of writes with low contention.
